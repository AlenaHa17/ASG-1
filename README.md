
# Crum - Fashion E-commerce Website

Crum is made to be a cute-style, responsive e-commerce website for a fashion brand specializing in soft, feminine clothing and accessories. The website features a clean, elegant and simplistic design with a focus on user experience and visual appeal.

# Design Process

## Target Audience
Crum targets girls and young women who are interested in cute, feminine, comfortable clothing with charming details. The design focuses on attracting customers through soft colors, elegant typography, and curated product presentations.

## User Stories
As a shopper, I want to
-	Browse products by category to find items I'm interested in
-	View product details including images, prices, and sizes
-	Add items to my shopping cart for later purchase
-	See what items are currently trending, new arrivals, top picks and limited edition
-	View outfit inspiration from real models/influencers
-	Easily navigate the site on both desktop and mobile devices

As a returning customer, I want to
-	See if the items I want are low on stock
-	See my previous shopping cart contents and total
-	Continue shopping where I left off

## Design Approach
The design follows a soft, feminine aesthetic with
- Color Palette: Soft pink, warm brown, and light gray for a gentle, inviting feel
- Typography: Playfair Display for elegance and Cormorant Garamond for readability
- Layout: Clean grid-based design with white space
- Imagery: High-quality product photos with model/celebrity coordination examples

# Features

## Existing Features
1. Responsive Navigation
-	Fixed header with logo and cart access
-	Dropdown category menu for easy browsing
-	Mobile-optimized hamburger menu for smaller screens

2. Product Showcase
-	Grid-based product display with hover effects
-	Detailed product pages with size selection
-	"New Arrivals" section highlighting latest products

3. Shopping Cart
-	Add/remove items with quantity adjustment
-	Persistent cart storage using localStorage
-	Real-time price calculation and summary

4. Visual Coordination
-	Model/celebrity coordination gallery for outfit inspiration
-	Hover zoom effects on product images
-	Clean and focused product photography

5. Responsive Design
-	Responsive layout for all screen sizes
-	Mobile-optimized touch interactions
-	Flexible image display with proper cropping

# Testing

## Manual Testing Scenarios

### Navigation
1.	Click all navigation links to ensure they go to correct pages
2.	Test dropdown menu on hover/tap
3.	Verify mobile menu appears correctly on small screens
4.	Check active state highlighting on current page

### Product Interaction
1.	Click product cards to view details
2.	Test size selection on product pages
3.	Add items to cart and verify quantity updates
4.	Test cart item removal and quantity adjustment

### Responsive Design
1.	Test layout on mobile (≤480px)
2.	Test layout on tablet (768px-1024px)
3.	Verify images scale appropriately

## Known Issues
1.	Image Loading: Some product images may not display if file paths are incorrect
2.	Cart Persistence: Cart clears if localStorage is cleared by browser
3.	Mobile Dropdown: Category dropdown requires click instead of hover on mobile
