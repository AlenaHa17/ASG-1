# Crum - Clothing Website

A minimalist clothing website inspired by Pium (clothing brand), specializing in cute skirts and tops.

## Features
- Clean, minimal design
- Responsive layout
- Smooth scrolling navigation
- Product hover effects

## Colors
- Soft pink & cream palette
- Earthy brown accents
- Light gray elements

## Pages
- Homepage with hero section
- New arrivals products
- Simple navigation
